# Logo Assets

## Logo File

Place your logo file here as `logo.png`.

## Recommended Specifications

- **Format**: PNG with transparency
- **Size**: 512x512px or higher (for retina displays)
- **Aspect Ratio**: Square (1:1) recommended
- **Background**: Transparent
- **File Size**: Optimized (under 100KB recommended)

## Usage

The logo is automatically used in:
- Header navigation (top left)
- Footer (bottom left)

The logo will be displayed at 32x32px (h-8 w-8) in both locations.

## Alternative Formats

If you have different logo formats, you can also add:
- `logo.svg` - Vector format (scalable)
- `logo-white.png` - White version for dark backgrounds
- `favicon.ico` - Browser favicon

## Next.js Image Optimization

The logo uses Next.js Image component which automatically:
- Optimizes image loading
- Provides lazy loading
- Supports responsive images
- Maintains aspect ratio
