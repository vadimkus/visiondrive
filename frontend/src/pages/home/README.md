# Home Page

## Purpose

Central hub; immediately establishes core service and value proposition.

## Page Sections

### Hero Section
- **High-impact video loop**: Reserved spot turning green animation
- **Headline**: "Guaranteed Parking. Seamless Mobility. Driven by Vision."
- **Subheadline**: Brief value proposition
- **Visual**: Engaging video/animation showing parking availability in real-time
- **CTAs**: 
  - Primary: "Download Our App" (for users)
  - Secondary: "Request Pilot Demo" (for partners)

### Core Value Section
- **Problem Statement**: Wasted time searching for parking
- **Solution Overview**: Real-time availability through smart technology
- **Key Benefits**: 
  - Guaranteed parking spots
  - Time savings
  - Reduced stress
  - Seamless experience

### How It Works (3 Steps)
1. **Sensor**: Detects vehicle presence
2. **Gateway**: Collects and transmits data
3. **App**: Displays real-time availability to users

**Visual**: Step-by-step diagram or animation showing the flow

### Featured Clients/Pilot Areas
- **Showcase**: Display the 20 pilot spots
- **Map View**: Interactive map showing pilot locations
- **Partner Logos**: Visual representation of partners
- **Success Metrics**: Early results from pilot program

### Trust Indicators
- **Statistics**: 
  - Number of parking spaces
  - Active users
  - Time saved
  - Pilot locations
- **Partner Logos**: RTA, commercial partners
- **Testimonials**: Early user feedback

### Call-to-Action Section
- **For Users**: "Download Our App" with app store badges
- **For Partners**: "Request Pilot Demo" with contact form link
