# Sensor Images

Place sensor product images here:

- `flush-mount-sensor.jpg` - Flush-mount parking sensor product photo
  - Recommended size: 800x600px
  - Format: JPG or PNG
  - Should show the sensor embedded/flush-mounted in parking surface

## Image Sources

You can find sensor images from:
- Sensor manufacturer websites
- Product catalogs
- Stock photo sites (with proper licensing)

The image will be displayed in the Technology Stack section of the Vision page.
